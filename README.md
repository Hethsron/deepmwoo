# DeepMwoo

## Welcome to **DeepMwoo**, Mwoo facial recognition solution !

## Features

*   Real-Time face tracking
*   Real-Time facial recognition
*   Facial character analysis
*   Facial motion capture

## Platforms

DeepMwoo has been used on a variety of plateforms:

*   Linux
*   macOS X
*   Windows
*   Raspberry Pi 3+
